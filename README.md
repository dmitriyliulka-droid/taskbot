# 🤖 TaskBot v3

Telegram бот для постановки задач з чек-поінтами, дедлайнами та автоматичними звітами.

---

## 👥 Ролі

| Роль | Що може |
|------|---------|
| **Керівник** | Ставити задачі будь-яким підлеглим, бачити свою статистику, отримувати авто-звіт |
| **Співробітник** | Отримувати задачі від будь-яких керівників, переглядати активні задачі, виконувати з коментарем |

> Один підлеглий може отримувати задачі від кількох керівників одночасно.

---

## 🚀 Запуск

```bash
npm install
cp .env.example .env
# Встав BOT_TOKEN в .env
npm start
```

---

## 📋 Команди

**Керівник:**
- `/manager <пароль>` — отримати права керівника
- `/menu` — головне меню
- `/stats` — статистика
- `/mytasks` — всі свої задачі

**Підлеглий:**
- `/start` — реєстрація
- `/mytasks` — мої активні задачі

---

## 🔄 Флоу задачі

```
1. Керівник → "Нова задача"
   → Вибирає підлеглого
   → Назва → Опис (опціонально)
   → Чек-поінти (кожен з нового рядка, опціонально)
   → Дедлайн → Нагадування → Підтвердити

2. Підлеглий отримує задачу:
   📬 Нова задача від [Керівник]!
   📌 Назва задачі
   📄 Опис
   Чек-поінти:
   1. Перший пункт
   2. Другий пункт
   📅 Дедлайн: 2024-12-31
   [✅ Виконано!]

3. Підлеглий натискає "Виконано" → пише коментар

4. Керівник отримує:
   ✅ Іван виконав задачу!
   📌 Назва задачі
   ☑️ Перший пункт
   ☑️ Другий пункт
   💬 Коментар: все зроблено, клієнт задоволений
```

---

## ⏰ Автоматика

- **Нагадування підлеглому** — через вибраний час якщо задача не виконана
- **Авто-звіт керівнику** — щодня о 18:00 (налаштовується)
- **Прострочені задачі** — підлеглі отримують нагадування щодня о 9:00
